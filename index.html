ハイライト英単語帳

英文を読みながら、知らない単語をその場で単語帳へ。
単語にマウスを乗せると意味がすぐ出ます（スマホはタップ）。
クリック/タップで「訳・使い方・例文」つきのカードとして単語帳に追加。
追加済みの単語は本文中でハイライトされます。 複数語のフレーズはドラッグで選択して追加できます。

学習設定

クラウド同期（Mac 2台・iPhoneでカードを共有）

English for Canada アプリと同じ方式です。全端末で同じURLと同期コードを入れると、単語帳と学習記録が自動で揃います（このアプリ専用の場所に保存されます）。

iPhoneの共有シートから追加（アプリを開かずに）

iPhoneのどのアプリでも「単語を選択 → 共有 → 単語帳に追加」で済むようになります。 「ショートカット」アプリで1回だけ次を作成:
  1. ショートカットアプリ → ＋ → 名前「単語帳に追加」
  2. ⓘ（設定）→「共有シートに表示」をオン（受け取る種類: テキスト）
  3. アクション追加:「URLの内容を取得」→ URLに下のアドレスを貼り付け
  4. 「方法」を POST に →「本文を要求」JSON → フィールド追加: キー w ・値 ショートカットの入力
（設定タブでクラウド同期を有効にしてください）
追加された単語は、次にどの端末でアプリを開いたときにも自動でカードになります。
おまけ: ページ丸ごと版（単語を選択すらしない）
もう1つショートカットを作り、共有シートの受け取りを「URL・Safari Webページ」にして、 アクションを「Webページから記事を取得（Safariリーダー）」→「URLの内容を取得（POST・上と同じURL）」 → 本文JSONのキーを text・値を「記事の本文」にします。
以後、記事を読み終えたら 共有 → そのショートカット。難しい単語だけが自動で「候補」に入り、 アプリを開いたときにまとめて確認・追加できます。

Macのどこからでも追加（アプリを開かずに）

初回のみ、アプリのフォルダにある 「セットアップ.command」をダブルクリック。 （アプリを更新したときも再実行してください）
① いちばん速い: ダブルコピー（⌘C を素早く2回）
どのアプリでも（Chrome・プレビューのPDF・メール…）英単語を選択して ⌘C ⌘C と2回コピーするだけ。即座に追加され、通知に訳が表示されます。 メニュー操作は一切ありません。
② ショートカット ⌘⇧E / 右クリック→サービス
選択して ⌘⇧E でも追加できます（効かないアプリは一度再起動を）。
③ Chrome拡張（選択ボタン＋Option+ホバー辞書）
chrome://extensions → 右上「デベロッパーモード」ON → 「パッケージ化されていない拡張機能を読み込む」→ このアプリのフォルダ内の chrome-extension を選択。
以後、単語を選択すると「＋単語帳」ボタンが浮かび、右クリックメニューの 最上部に「…を単語帳に追加」が直接出ます。 Option(⌥)+マウスオーバーならその場に訳がポップアップ、クリックで追加。

iPhoneなど（ブックマークレット）

SafariやChromeで読んでいるどんなWebページからでも、単語を選択→このブックマークを開くと、 このアプリが起動して単語帳に追加できます。
iPhone: 適当なページをブックマーク→ブックマーク編集でURLを上のコードに置き換え、名前を「単語帳に追加」などに。
Mac: ブックマークバーの適当なブックマークを編集して、URLを上のコードに置き換え。
以後、英単語を選択してそのブックマークを押すだけで追加画面が開きます。
